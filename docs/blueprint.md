# **App Name**: HakinzTech Digital Forge

## Core Features:

- Animated Hero Banner: Dynamic hero banner with animated text and graphics showcasing skills.
- Interactive Skill Display: Interactive skill showcase with animated progress bars and tool icons.
- Filterable Project Gallery: Filterable project gallery with smooth transitions and detailed case studies.
- Animated GitHub Stats: Animated statistics dashboard to showcase GitHub contributions and activity. Reads directly from the GitHub API.
- AI Project Summarizer: AI-powered tool that generates personalized summaries for projects based on their descriptions.  Acts as a TL;DR assistant for visitors, incorporating data as it reasons about project content and context.
- Interactive Contact Form: Interactive contact form with animated input fields and submission confirmation.
- Responsive Design: Fully responsive layout optimized for various screen sizes and devices.

## Style Guidelines:

- Primary color: Electric blue (#63B5FF) to convey innovation and energy.
- Background color: Dark gray (#2A2A2A) to create a modern and sophisticated feel.
- Accent color: Cyan (#40E0D0) for highlighting key elements and CTAs.
- Body and headline font: 'Space Grotesk' (sans-serif) for a modern, tech-focused aesthetic; falling back to 'Inter' for long body text
- Code font: 'Source Code Pro' (monospace) for displaying code snippets clearly.
- Use a consistent set of minimalist, geometric icons that complement the modern design.
- Implement subtle animations and transitions throughout the site to enhance user experience, such as parallax scrolling, hover effects, and loading animations.
